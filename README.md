# 🌍 LinguaLearn Pro

An interactive and modern Language Learning Web Application built using **HTML, CSS, and JavaScript**.

LinguaLearn Pro helps users learn new languages through flashcards, pronunciation practice, quizzes, progress tracking, and achievement badges in a clean and responsive interface.

---

## 🚀 Features

### 🌐 Multiple Language Support

Learn vocabulary from:

* 🇪🇸 Spanish
* 🇫🇷 French
* 🇩🇪 German

### 📚 Interactive Flashcards

* Flip cards to view translations
* Easy navigation with Previous/Next buttons
* Learn vocabulary in an engaging way

### 🔊 Pronunciation Support

* Built using the Web Speech API
* Listen to word pronunciations
* Improve speaking and listening skills

### 📝 Quiz Challenge

* Multiple-choice questions
* Instant feedback
* Score calculation
* Accuracy tracking

### 📈 Progress Tracking

* Visual progress bar
* Learning completion percentage
* Progress updates automatically

### 🔥 Learning Streak

* Track learning consistency
* Stay motivated with daily practice

### 🏆 Achievement Badges

Earn badges based on quiz performance:

* 🥉 Beginner
* 🥈 Intermediate
* 🥇 Expert

### 🌙 Dark Mode

* Modern dark theme
* One-click toggle
* User preference saved automatically

### 💾 Local Storage

* Saves theme preferences
* Stores learning progress
* Maintains user data between sessions

### 📱 Responsive Design

Optimized for:

* Desktop 💻
* Tablet 📱
* Mobile 📲
